# vue-zhihudaily

vue全家桶 学习项目
vue + vue router + vuex + webpack

欢迎star，fork，指正问题， 共同学习

使用方法

``` stata
git clone https://github.com/dclcats/vue-zhihudaily
cd vue-zhihudaily
npm install
npm run dev
```
注：node版本，我本机使用的版本为7.7.2 （建议使用yarn锁定第三方插件版本）
